# FocusRoom

**Study Together. Stay Focused.**

FocusRoom is a front-end prototype of a virtual co-working platform designed to help students and professionals stay productive through shared study spaces, focus sessions, and ambient work environments.

The idea behind FocusRoom is simple: working alongside others—even virtually—can improve concentration, consistency, and motivation.

---

## Overview

FocusRoom combines several productivity-focused features into a single interface:

- Virtual study rooms
- Pomodoro-based focus sessions
- Ambient sound controls
- Productivity analytics dashboard
- Interactive room management
- Responsive modern UI

This project was developed as a Web Technology prototype and serves as a proof of concept for a larger full-stack application.

---

## Features

### Virtual Study Rooms
Create, browse, and join study rooms based on different work styles.

- Deep Work rooms
- Exam Preparation rooms
- Collaborative rooms
- Custom room creation

### Focus Timer
Built-in Pomodoro timer designed for structured work sessions.

- Start / Pause / Reset
- Focus and break cycles
- Progress visualization
- Session tracking

### Ambient Sound Mixer
Create your preferred study environment using ambient sounds.

Available sound profiles include:

- Rain
- Ocean Waves
- Forest
- Fireplace
- Café Noise
- White Noise
- Keyboard Sounds
- Piano Notes

### Productivity Dashboard
Track focus activity through visual analytics.

- Focus hours
- Daily streaks
- Weekly progress
- Activity heatmap
- Room participation statistics

## Current Status

This project is currently a **prototype**.

The application demonstrates the user interface and core interactions, but it does not yet include:

- Database integration
- User authentication
- Real-time communication
- Backend APIs
- Persistent user data

These features are planned for future development.

Open the application:


Or simply double-click `index.html` in your file manager.
